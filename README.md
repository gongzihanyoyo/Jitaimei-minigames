## Jitaimei-minigames

### Jitaimei小游戏官方开源仓库

在线体验：[**minigames.jitaimei.top**](https://minigames.jitaimei.top/)

by Jitaimei Studio & JLI Tech Studio



本仓库提供Jitaimei小游戏网站中的**大多数**游戏的源代码。

Q：为什么是“大多数”？

A：本仓库中没有给出源代码的游戏，一般是有以下两种原因：

- 不是我们原创的。

- 由其他语言（如Scratch）转译过来的，文件体积巨大且一堆base64没有参考价值（后续可能考虑给出原语言的源代码）。
